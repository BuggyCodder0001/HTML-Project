HTML MEDIA

HTML Media Elements

1. The <audio> Tag

  • controls → Shows play, pause, volume controls.
  • autoplay → Starts playing automatically when the page loads.
  • loop → Plays repeatedly after finishing.
  • muted → Starts audio with sound muted.
  • preload → Informs browser how to load:
    • auto → Load whole audio file.
    • metadata → Load only info (length, size).
    • none → Don’t load until user clicks play.
  • src → Path of audio file (can also use <source>).
        
        <audio controls loop preload="metadata">
        <source src="song.mp3" type="audio/mpeg">
        <source src="song.ogg" type="audio/ogg">
        Your browser does not support audio.
        </audio>

2. The <video> Tag

  • controls → Shows video controls (play, pause, volume, fullscreen).
  • autoplay → Starts playing automatically when the page loads.
  • loop → Restarts automatically after finishing.
  • muted → Starts video with no sound.
  • poster → Thumbnail image before playing.
  • width / height → Sets the video size.
  • preload → Same as in audio.
  • src → Path of video file (can also use <source>).
        
        <video width="400" controls poster="thumbnail.jpg">
        <source src="movie.mp4" type="video/mp4">
        <source src="movie.ogg" type="video/ogg">
        Your browser does not support video.
        </video>

3. The <source> Tag

  • Used inside <audio> or <video>.
  • src → File path.
  • type → Media type (e.g., audio/mpeg, video/mp4).
        
        <audio controls>
        <source src="song.mp3" type="audio/mpeg">
        </audio>

4. The <track> Tag

  • Used inside <video> for subtitles or captions.
  • kind → Type of text track:
    • subtitles → Translation.
    • captions → Audio description.
    • descriptions, chapters, metadata.
  • src → Path to subtitle file (.vtt).
  • srclang → Language of track.
  • label → Track label (shown to user).
  • default → Sets default track.
        
        <video width="400" controls>
        <source src="movie.mp4" type="video/mp4">
        <track src="subtitles_en.vtt" kind="subtitles" srclang="en" label="English" default>
        </video>
